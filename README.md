# Priority List

Личный планировщик задач с учётом рабочей недели. Django + HTMX, работает локально.

## Возможности

- Задачи с источником, описанием, статусом и плановым/потраченным временем
- Статусы: **Доступна**, **Заблокирована (эта неделя)**, **Заблокирована (не эта неделя)**
- Маркер 40 часов — черта, после которой задачи не войдут в текущую неделю
- Эффективное время = запланировано − потрачено; заблокированные «не эта нед.» не занимают место в неделе
- Фильтры: Все / Активные / Заблокированные
- Сортировка перетаскиванием
- Редактирование двойным кликом по карточке
- Импорт встреч из Outlook по ICS-ссылке
- Тёмная и светлая тема (сохраняется в браузере)

## Запуск

```bash
cd priority_list
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

Открыть: http://localhost:8000

## Импорт из Outlook

Создай файл `.env` в корне репозитория:

```
OUTLOOK_ICS_URL=https://outlook.office365.com/owa/calendar/.../calendar.ics
```

Как получить ссылку в Outlook web: **Настройки → Просмотреть все параметры → Календарь → Общие календари → Опубликовать календарь** → выбрать «Можно просматривать все сведения» → скопировать ICS-ссылку.

После этого кнопка **📅 Из Outlook** импортирует встречи текущей недели (Пн–Вс). Повторный импорт не дублирует записи.

## Стек

- Python / Django
- SQLite
- HTMX — обновление карточек без перезагрузки страницы
- Sortable.js — сортировка перетаскиванием
- `icalendar` + `requests` — разбор ICS-календаря
